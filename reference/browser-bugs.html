# Browser Bugs

Reference the complete list of browser bugs related to the HiQ framework. Note that most of these are general bugs not directly tied to HiQ styles.

Bug Browsers
The details element is currently not supported in Edge. Edge
Inputs with display: inline-flex applied currently do not display correctly in a line in Edge. Edge
There are currently some issues styling the draggable button on range inputs in Edge, specifically related to sizing and border-radius. Edge
Applying button styling to file inputs is currently not supported in Edge. Edge
Indeterminate progress bar styling is currently not supported in Edge. Edge
